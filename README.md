# Atividade 1.2 Avaliativa de 2026.1 - 1o Bimestre - Sistemas operacionais

## Informações gerais

- **Público alvo**: alunos da disciplina de **Sistemas operacionais** do curso de [TADS](https://diatinf.ifrn.edu.br/cursos/tecnologia-em-analise-e-desenvolvimento-de-sistemas/) na [DIATINF](https://diatinf.ifrn.edu.br/) no [CNAT-IFRN](https://portal.ifrn.edu.br/campus/natalcentral/)
- **Professor**: [L A Minora](https://github.com/leonardo-minora/)
- **Objetivo**:
  1. Praticar programação com semáforos
  2. Implementar código para resolver os problemas de concorrência clássicos
     1. Produtor / Consumidor
     2. Leitores / Escritores
     3. Jantar dos filósofos
- **data de apresentação**: 03/07/2026

---
## Passos sugeridos da ativadade
1. Fork desse repositório
2. Verificar como na linguagem trata concorrência (regiões críticas)
   1. Verificar se existe biblioteca de exclusão mútua como semáfofo
   2. Verificar se existe estruturas de dados que tratam concorrência
3. Criar docker para executar o aplicativo
4. Criar e executar dentro do docker implementações para os problemas
   1. Criar o código com implementação de 1 problema
   2. Executar no docker e registrar o comportamento
   3. Se necessário, adaptar / modificar e executar novamente
5. Criar os slides para apresentação

**Lembre** de anotar suas dificuldades, curiosidades, dúvidas, entre outras coisas e acontecimentos do experimento.

---
## Guia de apresentação
Sequência de slides
1. Título (implementação de problemas clássicos de concorrência), linguagem e membros do grupo
2. Sumário
3. Bibliotecas usadas
4. Apresentar parte do código que trata a concorrência
5. Dificuldades
6. Demais observações do grupo
